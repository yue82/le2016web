準備

以下のファイルをダウンロードしてください．
  • def2.l
    flexプログラムdef2.lは，環境テスト(lex)で使用したflexプログラムdef.lに機能を追加したものです． "%%"で囲まれた部分に字句解析系のルールが記述されています．
  • test1.c
    テスト用プログラムtest1.cは，def2.lによる字句解析機能の一部を確認するプログラムです．
flexプログラムにテスト用プログラムを入力した際の動作は，環境テスト(lex)と同様に，以下のような手順で確認することができます． 
  $ flex def2.l
  $ gcc lex.yy.c
  $ ./a.out < test1.c
    IF LEFT_PARENTHESIS IDENTIFIER OPERATOR IDENTIFIER RIGHT_PARENTHESIS LEFT_BRACE TYPENAME IDENTIFIER OPERATOR INTEGER SEMICOLON IDENTIFIER OPERATOR IDENTIFIER SEMICOLON RIGHT_BRACE

問1

def2.lによる全ての字句解析機能が正しく動作していることを確認できるようなテスト用プログラムをtest1.cに追加してください． また，書いたテスト用プログラムを入力として，def2.lによる字句解析にかけた出力を示してください．
※注意
  • テスト用プログラムは，プログラムとして動作できる必要はありません．
  • 整数や識別子などについてはいくつかの例を作成し，解析系が様々な入力に対応していることを確認してください．

問2

del2.lの字句解析プログラムに，以下の予約語トークンを判別する機能を追加してください．(表記：トークン / 出力する文字列)
  • while / WHILE
  • do / DO
  • for / FOR
  • break / BREAK
  • continue / CONTINUE
追加した機能を正しく動作していることを確認できるようなテスト用プログラムをtest1.cに追加してください． また，書いたテスト用プログラムを入力として，def2.lによる字句解析にかけた出力を示してください．

問3

問2で作成したプログラムに，さらに以下のトークンを追加してください．(表記：トークン / 出力する文字列)
  • 2進数 / BINARY
  • 16進数 / HEXADECIMAL
  • 8進数 / BINARY
  • 浮動小数点数 / REAL
※注意
  • 浮動小数点数については，"1.2e+10"や"1.2e10"のような指数部を持つ値を認識できるようにして下さい．
  • C言語の仕様にならい，2進数の表記は"0b"から，16進数の表記は"0x"から，8進数の表記は"0"からはじまるものとします．
  • 0から始まる数字であっても，"020e+10"のような浮動小数点数の形式をしている場合は浮動小数点数として扱われることに注意してください．
追加した機能を正しく動作していることを確認できるようなテスト用プログラムをtest1.cに追加してください． また，書いたテスト用プログラムを入力として，def2.lによる字句解析にかけた出力を示してください．
